# UniVoice – Assistive Communication App

UniVoice is a cross-platform accessibility application designed to improve communication for people with hearing, speech, and visual impairments.

## Features
- 🎤 Speech-to-Text (STT)
- 🔊 Text-to-Speech (TTS)
- 🖐️ Sign Language Recognition
- 📷 Real-Time Object Detection
- 💵 Currency Detection
- 📱 Cross-platform Mobile Application

## Tech Stack
- React Native
- Python
- TensorFlow
- OpenCV
- MediaPipe
- FastAPI / Flask (Backend)

## Target Users
- Deaf users
- Mute users
- Visually impaired users

## Project Objective
To provide a single unified assistive communication platform that enables seamless interaction using AI-powered accessibility features.
